# 💰 Family Budget Tracker (Zen Style)

A comprehensive, mobile-first financial management system designed for families. Track expenses, manage savings goals, and monitor currency exchange rates in one unified, modern interface.

---

## 🇺🇸 English Version

### 🌟 Key Features

#### 🏦 Multi-Currency & Auto-Conversion
*   **Base Currency:** All accounting is centralized in **Moldovan Leu (MDL)**.
*   **Supported Currencies:** Full support for USD, EUR, and RON.
*   **Automatic Re-calculation:** Enter transactions in any currency; the system automatically converts them to MDL using real-time exchange rates.
*   **Live Rates:** Exchange rates are synchronized every 12 hours from a reliable API.

#### 👨‍👩‍👧‍👦 Family-First Management
*   **Collaborative Budgeting:** All family members work within a shared workspace, seeing the same balance, history, and categories.
*   **Activity Ownership:** Every transaction and system action is tagged with the user's name, providing clear visibility on who added what.
*   **Admin Controls:** Dedicated dashboard for administrators to create/delete users and manage system-wide settings.

#### 📈 Smart Planning & Analytics
*   **Budget Limits:** Set monthly spending targets for categories. Visual progress bars turn red when you approach your limit.
*   **Savings Goals:** Define goals (e.g., "Home Deposit", "Summer Trip"), update collected amounts, and track progress percentages.
*   **Interactive Timeline:** Transactions are elegantly grouped by day with automatic daily totals.
*   **Powerful Search:** Instantly find any transaction using the built-in search bar and date range filters.

### 🛠 Tech Stack
*   **Frontend:** React (TypeScript), Vite, Chart.js, Lucide Icons, i18next.
*   **Backend:** Node.js, Express (TypeScript), JWT (Authentication), Bcrypt (Security).
*   **Database:** PostgreSQL.
*   **Deployment:** Docker & Docker Compose.

### 🚀 Getting Started
1. Ensure **Docker** and **Docker Compose** are installed.
2. Run the following command:
   ```bash
   docker-compose up --build
   ```
3. Open your browser at **http://localhost**

## 🇷🇺 Русская версия

### 🌟 Основные возможности

#### 🏦 Мультивалютность и Авто-конвертация
*   **Базовая валюта:** Все расчеты ведутся в **молдавских леях (MDL)**.
*   **Поддержка валют:** USD, EUR, RON.
*   **Автоматический пересчет:** Вносите траты в любой валюте, система сама пересчитает их в леи по актуальному курсу.
*   **Актуальные курсы:** Синхронизация данных каждые 12 часов.

#### 👨‍👩‍👧‍👦 Семейный доступ
*   **Общее пространство:** Все члены семьи видят общие счета, категории и историю операций.
*   **Прозрачность:** В ленте транзакций и активности всегда видно, кто именно внес изменения.
*   **Администрирование:** Удобная панель управления пользователями и настройками регистрации.

#### 📈 Аналитика и Планирование
*   **Лимиты по категориям:** Устанавливайте бюджет на месяц. Индикаторы подскажут, когда вы близки к превышению.
*   **Копилки:** Создавайте цели для накоплений и отслеживайте прогресс в процентах.
*   **Удобная лента:** Группировка операций по дням с промежуточными итогами.
*   **Поиск и Фильтры:** Мгновенный поиск по описанию и фильтрация за любой период времени.
